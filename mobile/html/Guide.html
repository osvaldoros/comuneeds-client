Guia

El proceso sugerido para llevar a cabo la evaluacion de necesidades es el siguiente:

1) Crear una matriz para capturar los respuestas de las comunidad para cada necesidad

Se puede crear una matriz de elementos destructivos (que dificultan la satisfaccion de las necesidades) o una matriz de factores ideales para satisfacer cada necesidad, tambien puedes crear matrices en diferentes momentos y compararlas o cualquier otro uso que tu les quieras dar!

2) Crear equipos de trabajo (aprox 10 pnas)

3) Cada equipo se reune por separado y discuten entre sus miembros para ingresar los elementos que consideren importantes en cada categoria de cada necesidad

El coordinador del equipo es responsable de ingresar la informacion al sistema

4) Una vez recibidas todas las respuestas de los equipos, el administrador unifica las respuestas.

El sistema automaticamente elimina duplicados y sinonimos.

5) A cada equipo se le asigna una de las 9 necesidades (ya unificadas) y de nuevo discutiendo entre ellos deciden el elemento mas importante y decisivo para cada una de las 4 categorias de esa necesidad

Este proceso sintetiza la matriz, para que quede un solo elemento en cada casilla

6) Se repite el proceso las veces que se desee para obetener diferentes matrices
